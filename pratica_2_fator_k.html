Medidor Turbina e Fator K

Processamento Digital de Pulsos em Medidores de Vazão Volumétrica

Sensor: Tipo Turbina (Pulsos) Sinal: Frequência (Hz) / Onda Quadrada Fator K: 450 pulsos/L Plataforma: Arduino + Gerador de Funções

🎯 1. Objetivos de Aprendizagem

  • Familiarizar-se com medidores de vazão do tipo turbina.
  • Programar o microcontrolador para ler sinais digitais em alta velocidade utilizando Interrupções de Hardware.
  • Explicar a importância do Fator K na calibração de medidores industriais.
  • Transformar a frequência dos pulsos (Hz) em vazão volumétrica (L/min).

📖 2. Fundamentos Teóricos

2.1 O Medidor Tipo Turbina

O medidor de turbina possui hélices que giram com a passagem do fluido. Embutido ou externo ao tubo há um sensor magnético (ou de Efeito Hall). A cada vez que uma pá da hélice passa pelo sensor no corpo do medidor, ele gera um pulso elétrico. O número de pulsos gerados é proporcional ao volume de fluido que passou pelo medidor.

2.2 O Fator K

O Fator K é a constante de calibração fornecida pelo fabricante do medidor (impressa na plaqueta do invólucro). Ele indica quantos pulsos eletromagnéticos o medidor produzirá para a passagem de 1 unidade de volume do meio medido.

No nosso laboratório, o Fator K estabelecido é de 450 pulsos / Litro. Assim, se passarem 450 pulsos, sabemos que passou exatamente 1 Litro de água na tubulação.

2.3 Equação de Conversão HZ → L/min

Sabemos que a Frequência ($f$) em Hertz (Hz) equivale ao número de pulsos por segundo.

// 1. Converter Frequência para Pulsos por Minuto
Pulsos/min = Frequência (Hz) × 60

// 2. Encontrar a Vazão usando o Fator K
Vazão (L/min) = Pulsos/min ÷ Fator K

2.4 Interrupções de Hardware (attachInterrupt)

O Arduino seria muito lento para ler a mudança de pinos com precisão de microssegundos usando apenas o digitalRead(). Por isso, usa-se a interrupção. A função attachInterrupt() força o processador a parar imediatamente o que está fazendo, rodar uma pequena Rotina de Serviço (ISR) para contar o pulso, e então continuar o código principal.

💻 4. Código Arduino para Contagem de Pulsos

#include <LiquidCrystal.h>

LiquidCrystal lcd(12, 11, 6, 5, 4, 3);

const int PINO_PULSO = 2; // Pino que suporta interrupção
const float FATOR_K = 450.0; // 450 pulsos/L

// 'volatile' indica ao compilador que ela muda em backgroud
volatile unsigned long contadorPulsos = 0;
unsigned long tempoAnterior = 0;

void contarPulso() {
  contadorPulsos++;
}

void setup() {
  lcd.begin(16, 2);
  pinMode(PINO_PULSO, INPUT_PULLUP);
  
  // Ativa Interrupção na borda de SUBIDA do sinal
  attachInterrupt(digitalPinToInterrupt(PINO_PULSO), contarPulso, RISING);
}

void loop() {
  unsigned long tempoAtual = millis();
  
  // A cada 1 segundo (1000 ms), fecha a janela de leitura
  if (tempoAtual - tempoAnterior >= 1000) {
    noInterrupts(); // Pausa p/ não corromper a leitura
    unsigned long pulsosLidos = contadorPulsos;
    contadorPulsos = 0;
    interrupts();
    
    tempoAnterior = tempoAtual;

    float frequencia_Hz = (float)pulsosLidos;
    float vazao_Lmin = (frequencia_Hz * 60.0) / FATOR_K;

    lcd.setCursor(0, 0);
    lcd.print("Freq: ");
    lcd.print(frequencia_Hz, 0);
    lcd.print(" Hz  ");

    lcd.setCursor(0, 1);
    lcd.print("Vazao:");
    lcd.print(vazao_Lmin, 2);
    lcd.print(" L/min ");
  }
}
