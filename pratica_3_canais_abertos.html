Cálculo de Vazão por Variação de Nível com Ultrassom Estilo CAESB

Sensor: Ultrassônico HC-SR04 Tanque: Cilíndrico (R = 0,5m; H = 1,0m) Sistema: Medição Indireta por ΔV/Δt Plataforma: Arduino + Tinkercad

🎯 1. Objetivos de Aprendizagem

  • Entender o princípio de medição indireta de vazão comum em saneamento básico (ETAs e ETEs).
  • Programar o HC-SR04 para medir o nível da água a partir do topo do reservatório.
  • Aplicar fórmulas de Geometria Espacial (Volume do Cilindro) no Arduino.
  • Calcular taxas de escoamento (L/s) comparando janelas de tempo ($Q = \Delta V / \Delta t$).

📖 2. Fundamentos Teóricos

2.1 Cenário de Aplicação (Ex: CAESB)

Em empresas de saneamento (como a CAESB, SABESP, etc), é muito comum não se ter um medidor de vazão direto em formato de tubo em todas as saídas de grandes reservatórios rurais apoiados. Para medir o escoamento, instala-se um instrumento ultrassônico de nível no topo. Sabendo a área do tanque, o CLP calcula quantos litros saíram olhando a diferença no nível da água ao longo de alguns segundos.

2.2 O Nível a partir do Ultrassom

O sensor ultrassônico mede distância: do topo do reservatório até a superfície da água. Logo, o nível de água ($h$) é:

Nível da Água (h) = Altura Total do Tanque − Distância Medida

2.3 Volume do Cilindro

Um reservatório cilíndrico tem o volume dado pela área da base multiplicada pela altura de líquido.

Área da Base (m²) = π × Raio (m)²
Volume (m³) = Área da Base × Nível de Água (m)
Volume (Litros) = Volume (m³) × 1000
No nosso laboratório: Raio = 0,5 metros. Altura do Tanque = 1,0 metro.

2.4 Vazão de Escoamento

A taxa de vazão volumétrica ($Q$) é a quantidade de fluido subtraída ao longo do tempo. Analisamos de $5$ em $5$ segundos:

Vazão (L/s) = (Volume Anterior − Volume Atual) ÷ Delta de Tempo (s)

Se o volume anterior for maior que o atual, significa que o tanque esvaziou, produzindo uma vazão de saída (positiva).

💻 4. Código Arduino Explicado

#include <LiquidCrystal.h>

LiquidCrystal lcd(12, 11, 5, 4, 3, 2); 

const int TRIG_PIN = 9;
const int ECHO_PIN = 10;

// Constantes Físicas do Tanque
const float ALTURA_TANQUE_M = 1.00; // 1 metro
const float RAIO_M = 0.50;          // 50 cm
const float AREA_TANQUE_M2 = 3.14159 * (RAIO_M * RAIO_M);

float volumeAnterior_L = 0.0;
unsigned long tempoAnterior = 0;
const unsigned long INTERVALO = 5000; // 5 segundos

// Função auxiliar para ultrassom (retorna em metros)
float medirDistanciaMetros() {
  digitalWrite(TRIG_PIN, LOW); delayMicroseconds(2);
  digitalWrite(TRIG_PIN, HIGH); delayMicroseconds(10);
  digitalWrite(TRIG_PIN, LOW);
  long duracao = pulseIn(ECHO_PIN, HIGH);
  float dist_cm = (duracao * 0.0343) / 2.0;
  return dist_cm / 100.0; 
}

void setup() {
  lcd.begin(16, 2);
  pinMode(TRIG_PIN, OUTPUT);
  pinMode(ECHO_PIN, INPUT);

  // Inicializa volume base para iniciar as subtrações correntes
  float nivel_m = ALTURA_TANQUE_M - medirDistanciaMetros();
  if(nivel_m < 0) nivel_m = 0;
  volumeAnterior_L = AREA_TANQUE_M2 * nivel_m * 1000.0; 
}

void loop() {
  unsigned long tempoAtual = millis();

  // 1. Nível Contínuo
  float nivelAtual_M = ALTURA_TANQUE_M - medirDistanciaMetros();
  if (nivelAtual_M < 0) nivelAtual_M = 0;
  float volumeAtual_L = (AREA_TANQUE_M2 * nivelAtual_M) * 1000.0;

  lcd.setCursor(0, 0);
  lcd.print("Nivel: "); lcd.print(nivelAtual_M * 100.0, 1); lcd.print(" cm  ");

  // 2. Vazão calculada em janelas fixas (5s)
  if (tempoAtual - tempoAnterior >= INTERVALO) {
    float deltaTempo_s = (tempoAtual - tempoAnterior) / 1000.0;
    float deltaVolume_L = volumeAnterior_L - volumeAtual_L;
    
    float vazao_Ls = 0.0;
    if (deltaVolume_L > 0) vazao_Ls = deltaVolume_L / deltaTempo_s;

    lcd.setCursor(0, 1);
    lcd.print("Vaz: "); lcd.print(vazao_Ls, 2); lcd.print(" L/s  ");

    volumeAnterior_L = volumeAtual_L;
    tempoAnterior = tempoAtual;
  }
}
